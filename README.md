# ClipFeed

**Stop copying the same things again and again.**

ClipFeed is a simple clipboard timeline that lets you **reuse things you copied before.**

Instead of copying the same text repeatedly — links, prompts, messages, or anything else — ClipFeed keeps them ready so you can paste them instantly.


The main application source code is private,
but releases and update metadata are managed here.

**Download:** [Releases](https://github.com/c-c-meguchan/clipfeed-site/releases)


---

## Why ClipFeed?

We all copy the same things every day.

* links
* prompts
* messages
* emails
* pieces of text

Normally you copy them again each time you need them.

ClipFeed lets you **copy once and reuse anytime.**

---

## How It Works

1. Copy something like usual (`Cmd + C`)
2. ClipFeed keeps it in your clipboard history
3. Paste it again instantly when you need it

That's it.

No complicated setup.



If macOS shows a security warning:

1. Right-click the app
2. Click "Open"
3. Click "Open" again

---

### Privacy by Design

ClipFeed is built as an offline-first app.
Your clipboard history never leaves your Mac.
No data is sent to any server.


---

## What People Use It For

* Reusing AI prompts
* Sending repeated messages
* Saving links you paste often
* Keeping frequently used text ready

---

## Philosophy

ClipFeed is intentionally minimal.

No heavy interface.
No complicated features.

Just a faster way to reuse things you copy.
